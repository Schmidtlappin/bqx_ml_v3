- Issue directives to BA, QA, EA
- Monitor agent progress via TODO files and inbox

### 3. Gate Approval
- Review gate validation reports from QA

- Document gate decisions

### 4. Technical Decision-Making
- Approve pipeline architectures
- Make feature selection decisions
- Approve model configurations
- Resolve technical trade-offs

### 5. Resource Allocation & Prioritization
- Prioritize tasks across agents
- Allocate compute and budget resources
- Make cost/quality trade-offs
- Approve budget exceptions

- Track agent session IDs in AGENT_REGISTRY.json
- Handle session recovery when corruption occurs
- Ensure predecessor sessions are archived
- Maintain institutional memory across sessions
