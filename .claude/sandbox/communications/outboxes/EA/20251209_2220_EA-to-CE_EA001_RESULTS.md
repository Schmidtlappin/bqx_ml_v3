## Root Cause Analysis

### ElasticNet Performance

| Metric | Value | Expected | Status |
|--------|-------|----------|--------|
| AUC | 0.4578 | ≥ 0.50 | **ANOMALY** |
| vs Other Models Avg | -0.3876 | Similar | **SEVERE GAP** |

### Root Causes Identified

| # | Cause | Severity | Explanation |
|---|-------|----------|-------------|
| 1 | AUC below random | HIGH | Predictions inversely correlated with truth |
| 2 | Performance gap | HIGH | 0.4578 vs 0.8454 (others avg) |
| 3 | Linear model limitation | MEDIUM | Forex features are highly non-linear |
| 4 | Feature scaling sensitivity | MEDIUM | Mixed scales may degrade performance |

### Primary Root Cause

**Linear model inappropriate for non-linear forex features**

ElasticNet assumes linear relationships between features and target. Forex momentum prediction involves complex non-linear patterns that tree-based models (LightGBM, XGBoost, CatBoost) capture effectively, but linear models cannot.
